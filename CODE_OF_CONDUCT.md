# Code of Conduct

## Our pledge

This project is a welcoming, harassment-free space for everyone, regardless of
experience level, background, or identity. Everyone interacting in the project's
issues, pull requests, and discussions is expected to be respectful and
considerate.

## Expected behavior

- Be respectful and constructive in discussion and feedback.
- Assume good faith; people contribute on their own time.
- Focus on what is best for the project and its users.
- Gracefully accept constructive criticism.

## Unacceptable behavior

- Personal attacks, insults, or derogatory comments.
- Harassment of any kind, public or private.
- Publishing others' private information without permission.
- Any conduct that would reasonably be considered inappropriate in a
  professional setting.

## Scope

This Code of Conduct applies within all project spaces — the issue tracker,
pull requests, and any other official communication channel — and when an
individual is representing the project in public spaces.

## Enforcement

Instances of abusive or otherwise unacceptable behavior may be reported to the
project maintainer via GitHub. All reports will be reviewed and handled with
discretion. The maintainer may take any action deemed appropriate, including
warning the offender or blocking them from the project's spaces.

---

This Code of Conduct is adapted from common open-source community guidelines.
